    4.编程介绍（二）

    Last Update: 2022/5/28

    本节将介绍几种常见编程语言。


    C语言：万物之源

    C语言诞生于1972年，用于构建Unix操作系统，偏向计算机底层操作，多用于性能要求严格的领域。几乎所有现代编程语言都脱胎于C。

    用途：

    操作系统

    Unix、Unix应用程序

    嵌入式领域、硬件开发

    C编译器、解释器、微控制器

    微波炉里的程序

    服务器领域

    网络核心设备（路由器、交换机、防火墙）

    软件开发

    计算器、文本编辑器


    C语言和其他语言的类似关系：

    法语写的小说>>>日译版>>>翻译日译版而来的其他译版

    机器/汇编语言>>>C语言>>>其他语言


    C++：难学的语言

    诞生于1983年，作为C语言的增强方案和升级版本。C++是C语言的超集，C语言的大部分知识也适用于C++。C++是面向对象（OOP）语言。C++允许大家对自己的应用进行灵活调整并充分发挥计算机的全部性能。与Java/C相比，编程过程并不友好，但我们能利用它实现一切操作。

    用途：

    操作系统

    Windows UI、MacOS UI等。

    搜索引擎

    Google：汇编与C++

    软件开发

    Office全家桶、eBay、FaceBook、PhotoShop、MediaPlayer

    IE浏览器、火狐浏览器、Chrome、Safari、Opera

    视频游戏

    星际争霸、魔兽世界、CS、跑跑卡丁车

    关系型数据库

    Oracle、MySQL、SQL Server等

    其他

    三维模型开发、图形处理、多种编译器


    Python：易学的语言

    诞生于1991年，一款服务器端的解释型脚本语言、面向对象、开源、社区活跃，被称为胶水语言，哪里都可以用。

    用途：

    Web开发

    YouTube、Instagram、美国中情局网站、豆瓣

    爬虫（Python是Google的官方语言之一）

    数据分析

    美国航天局（NASA）数据分析

    机器学习、AI

    图像识别、语音识别、无人驾驶等


    PHP：最好的语言（梗）

    诞生于1994年，一款服务器端脚本语言，最初表示个人主页（Personal Home Page），语法和C类似，有很多的模板和框架，简单易懂，上手非常快。

    用途：

    网络开发

    FaceBook、百度、新浪、Wiki

    API接口开发

    广告系统开发

    游戏开发



    梗来源1：编程语言鄙视链 C > C++ > Java > C# > JavaScript > VB > Python > PHP > All

    梗来源2：官方曾在PHP官网上描述称PHP是世界上（开发web）最好的语言


    Java：需求过旺的语言

    创建于1995年，是一种面向对象、基于类的编程语言。Java继承了C++，但抛弃了C++的一些缺点。拥有全球最大的开发者专业社群，API接口非常丰富，具有强大的开源开发工具，大量的开源共享库。（Java由Sun公司开发，后被Oracle公司收购）

    用途：

    服务端应用开发、分布式系统

    Web应用后台的首选语言

    大到全国联网的系统、淘宝、12306、银行证券系统等，小到中小企业的应用解决方案

    安卓移动端开发：混合开发（Java、HTML+CSS+JavaScript）

    大数据开发

    游戏开发

    我的世界

    桌面应用开发


    当前Java主要应用方向：

    服务端编程

    网络编程

    企业常用Java开发工具

    Eclipse（IBM做的，取名寓意干掉Sun公司）

    Intellij IDEA（捷克的JetBrains公司开发出的，其旗下每一款编译器都是爆火产品。）

    Android Studio（谷歌做的，支持Java和Kotlin开发安卓应用。Kotlin是谷歌开发，专门用于开发安卓的语言。据说使用上还是不如Java。）


    JavaScript：前端为王

    诞生于1995年，是目前所有主流浏览器唯一支持的脚本语言。在前端开发中，JS占有不可替代的地位。

    前端三剑客：JavaScript、HTML、CSS（后二者是标记语言，严格上来说不是一种编程语言）

    用途：

    网站前端

    服务器端开发

    基于Node.js用于服务器端的开发



    在浏览器中打开任何一个网页页面，按F12可查看该页面的源码，然后Ctrl+Shift+C可“指哪看哪”。也可在页面点击右键，“查看网页源码”


    C#：背靠大树的语言

    诞生于2000年，一款强大而灵活的编程语言。原本曾被定名为Cool（C-like Object Oriented Language），由于商标原因更名为C#。由于靠着微软这颗大树，所以可以充分发挥微软.NET平台的优势，也是多年来windows平台的一门主流编程语言。

    用途：

    Windows桌面应用开发

    企业级业务应用

    软件开发

    App开发

    游戏开发（Unity）
